amesh
==========

みんな大好き東京アメッシュ http://tokyo-ame.jwa.or.jp/
をCLIで表示

![](https://cloud.githubusercontent.com/assets/931554/11038037/5940e5be-8744-11e5-94d9-4b0bc7b2f55f.png)

install
==========

```
go get github.com/otiai10/amesh/amesh
```

usage
==========

```sh
amesh     #降雨状況のみ描画
amesh -g  #地形情報を描画
amesh -m  #地名情報を描画
```
